## Objective
Build a single-page marketing website for Progression Labs that serves as a USPTO trademark specimen for two marks: "Progression Labs" (Class 35 & 42) and "WeAreProgression" (Class 35 & 42). The page must look like a real, operating business website.

## Architecture
- Static single-page site (HTML + CSS + JS)
- No build tools or frameworks
- Inter font from Google Fonts
- GSAP + ScrollTrigger from CDN for animations
- Deployable to any static host

## Work Breakdown

1. **Logo SVG** - Circle-grid P logomark (~small)
   - Files: logo.svg
   - Verification: Renders correctly in browser

2. **CSS Design System** - Full stylesheet with brand tokens (~medium)
   - Files: styles.css
   - Verification: Color palette, typography, responsive breakpoints correct

3. **HTML Structure** - Complete page with all 9 sections (~large)
   - Files: index.html
   - Verification: All trademark requirements met, both marks visible

4. **GSAP Animations** - Scroll-triggered professional animations (~medium)
   - Files: main.js
   - Verification: Animations subtle and professional

5. **Deploy & Verify** - Push to GitHub, verify checklist (~small)
   - Verification: All checklist items pass

## Critical Legal Requirements
- [x] "Progression Labs" clearly visible (nav, hero, footer)
- [x] "WeAreProgression" clearly visible (footer, social handle)
- [x] "Log in" button in navigation (exact text)
- [x] Class 35 services described (business consultancy, digital transformation, advisory)
- [x] Class 42 services described (AI technology, software, AIaaS, technical support)
- [x] USD pricing visible
- [x] Page looks like a real business website
- [x] Footer: (C) 2026, contact email, legal links
